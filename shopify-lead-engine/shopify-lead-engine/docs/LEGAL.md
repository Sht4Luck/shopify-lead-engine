## What This Software Does

### Data Collection Methods

This software collects ONLY publicly available information:

1. **Web Search**: Uses public search engines (Bing)
2. **Public Websites**: Scrapes only publicly accessible pages
3. **Contact Information**: Extracts only information displayed publicly
4. **No Authentication**: Never accesses password-protected content
5. **No Social Media Scraping**: Does not scrape Instagram, Facebook, etc.

### Technical Limitations

This software:
- Cannot access private or logged-in content
- Cannot bypass CAPTCHAs or security measures
- Cannot send emails or messages automatically
- Cannot access databases or admin panels
- Respects robots.txt directives (via Playwright)
